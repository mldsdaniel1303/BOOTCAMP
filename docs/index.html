Captura de Tela Rápida

Uma extensão do Chrome (Manifest V3) para capturar screenshots da aba ativa e baixar a imagem com escolha de local.

Funcionalidades

  • Capturar screenshot da aba visível atual
  • Baixar como PNG com timestamp no nome
  • Diálogo de salvamento para escolher o local no PC

Screenshots

Instalação (Modo Desenvolvedor)

  1. Baixe o ZIP da extensão.
  2. Abra chrome://extensions no Chrome.
  3. Ative o Modo Desenvolvedor.
  4. Clique em Carregar sem compactação e selecione a pasta da extensão descompactada.

Configuração para Escolha de Local

No Chrome: Configurações > Downloads > Marque "Perguntar onde salvar cada arquivo antes de baixar".

Permissões

  • tabs: Para capturar a aba ativa
  • downloads: Para baixar o screenshot
  • activeTab: Para interagir com a aba atual
  • https://*/*, http://*/*: Para funcionar em todas as páginas web
